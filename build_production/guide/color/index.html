Variantes de Color

Clases para aplicar colores de texto (`clN`) y fondo (`bgN`) fijos (1 a 10), útiles para acentos visuales específicos.

Fondos de Variantes (`bgN`)

Aplica el color de fondo y el color de texto de contraste predefinido.

bg1 (Teal)
bg2 (Soft Coral)
bg3 (Muted Gold)
bg4 (Slate Blue)
bg5 (Olive Green)
bg6 (Mauve)
bg7 (Soft Orange)
bg8 (Dusty Pink)
bg9 (Soft Cyan)
bg10 (Muted Brown)
